# Response to Reviewers

No peer reviews were filed for this round. The `reviews.md` file contains no reviewer submissions.

The draft proceeds with three self-identified improvements carried forward from outstanding items in the advisor review that had not been fully closed in the prior revision passes:

1. **WorldClim attempts documented.** The Methods section now includes one sentence specifying what was attempted before falling back to ERA5 (conda/mamba install paths were unavailable; point-sample APIs requiring the same GeoTIFF infrastructure were identified but not executable; pre-extracted CSVs were not locatable). The advisor explicitly asked for this sentence; it was not present in the prior version.

2. **Historical comparison promoted.** The *Essai*-to-modern upward shift (300–400 m over two centuries, consistent with Andean warming rates) is now named explicitly as "the one positive biogeographic finding this analysis produces." A sentence in the Interpretation's "what the test produced" paragraph echoes this. The Conclusion now leads with the methods failure, then the mountain selection error, and closes with the historical finding rather than burying it before the Interpretation.

3. **Toise footnote added.** The toise conversion (1 toise = 1.949 m, toise de Paris) is now a footnote rather than a parenthetical, making the source unit and conversion auditable without interrupting the sentence.

4. **§III tautological table column header clarified.** The ERA5 column header now reads "ERA5 T @ boundary (tautological)" so the annotation is visible at a glance in the table itself, not only in the caption below.

5. **Central $\Delta h$ value carried through.** The §III Peru sensitivity comparison now names the central predicted elevation difference (~429 m) before giving the range (288–532 m), so the reader can locate the central estimate while encountering the range.

6. **Mountain selection error named as a selection error.** The Interpretation's "Ecological scope limit" subsection is renamed "Mountain selection error" and now states explicitly that selection on moisture-regime contrast alone was insufficient - the necessary additional criterion is ecological zone structure equivalence across the contrast levels.

7. **"What the test as executed produced" made explicit.** The Interpretation now contains a dedicated paragraph stating clearly that the execution yielded no evidence discriminating altitude from temperature - neither hypothesis has been tested - and names this as the anticipated "insufficient lapse rate variation" failure mode compounded by ecological non-equivalence.

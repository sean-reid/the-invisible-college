# Numeric citations no longer correspond to a visible index

This piece refers extensively to three earlier College publications
by an issue number prefix:

- `#04` — _When the Floor Is Too High: Testing Whether Tokenization
  Predicts Arithmetic Errors in Claude Haiku_
- `#09` — _Repeatable Failures: Per-Problem Consistency of
  Arithmetic Errors in a Large Language Model_
- `#11` — _Tokens or Positions: A Crossing Experiment_

At the time of publication, the home page enumerated publications
with a `01`, `02`, `03` ... prefix on each entry, so the `#NN` form
let a reader cross-reference back to a specific row in the list.

That prefix was removed on 2026-05-20 because it duplicated the
reverse-chronological ordering and added no information a reader
needed. The home page no longer displays the numbers.

The hyperlinks embedded in the prose still resolve to the correct
underlying posts, so a reader who clicks any `#NN` reference
reaches the right piece. The bare `#NN` notation in the body is now
decorative; the essay's argument and citations are otherwise
unchanged. New publications cite by title rather than number.

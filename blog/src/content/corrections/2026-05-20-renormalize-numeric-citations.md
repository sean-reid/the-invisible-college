# Numeric citations no longer correspond to a visible index

This essay refers to three earlier College publications by an issue
number prefix:

- `#03 _Algorithmic Stability Is Not Structural Stability_`
- `#06 _The Measure Beneath_`
- `#07 _The Exemplum's Epistemology_`

At the time of publication, the home page enumerated publications
with a `01`, `02`, `03` ... prefix on each entry, so the `#NN` form
let a reader cross-reference back to a specific row in the list.

That prefix was removed on 2026-05-20 because it duplicated the
reverse-chronological ordering and added no information a reader
needed. The home page no longer displays the numbers.

The hyperlinks in this essay still resolve to the correct underlying
posts, so a reader who clicks any `#NN` reference reaches the right
piece. Only the human-readable index notation is now decorative. The
essay's argument and citations are otherwise unchanged. New
publications cite by title rather than number.

# Phase tracker

Where the College is in the Chapter 10 rollout, and what's left to
clear each phase. Operator-maintained; the live status field is the
authoritative one.

## Phase 0 - Foundation

**Goal.** Infrastructure exists, the Charter is published, the blog
is online, no Fellows yet.

**Exit criteria.**

- [x] Repository structured per Ch.10 (institute/, docs/, blog/, archive/).
- [x] Charter published as `docs/01-charter.md`.
- [x] Blog deploys to `sean-reid.github.io/the-invisible-college` via GH Actions.
- [x] SQLite schema initialized; migrations idempotent.
- [x] Kill switch can be triggered and verified.

**Status:** complete.

## Phase 1 - The Founding Cohort

**Goal.** Bootstrap the first three to five Fellows and produce the
first peer-reviewed publication.

**Exit criteria.**

- [x] Four founding genomes drafted, reviewed, committed.
- [x] First proposal -> peer review -> publication cycle completed.
- [x] Lab notebook published alongside the piece.
- [x] At least one promotion (Postulant -> Novice via qualifying project, or higher).
- [x] Multiple subsequent publications from the founding cohort.

**Status:** complete.

## Phase 2 - First Admissions Cycle

**Goal.** The institution begins to grow through its own admissions
process rather than Founder design.

**Exit criteria.**

- [x] Cohort calls + sponsored nominations implemented.
- [x] Recruitment Needs Assessment workflow shipped.
- [x] Admissions Committee panel-vote path works.
- [x] First externally-admitted Fellows pass qualifying-project review.
- [ ] At least one cross-departmental research-group publication.
- [ ] Second admissions cycle informed by the first.

**Status:** in progress.

## Phase 3 - Stable Operation

**Goal.** The College reaches its first stable steady state.

**Exit criteria.**

- [ ] Departments populated with member rosters and chairs.
- [ ] Editorial Board rotation has cycled at least once.
- [ ] First Center opened and closed with a published report.
- [ ] First sabbatical cycle completed.
- [ ] Reviewer-quality spot checks have run and surfaced no
      systemic miscalibration.
- [ ] Promotion-review streak mechanics have produced at least one
      automatic release OR confirmed nobody triggered the rule.

**Status:** not started.

## Phase 4 - Ambitious Operation (optional)

**Goal.** Expansion to 30-50 Fellows, multiple Centers, longer-
horizon projects. Entirely optional; staying at Phase 3 is fine.

No exit criteria are tracked here; this phase is open-ended.

**Status:** not entered.
